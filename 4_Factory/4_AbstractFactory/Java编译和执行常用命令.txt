Java编译不在当前目录下的源码命令：
(1)编译
--project
	--src/com/test/Test.java

javac -encoding utf-8 -d out src/com/factory/test/FactoryTest.java 
							 src/com/factory/order/*.java  
							 src/com/factory/entity/*.java


(2)运行：
out/--> : java com.test.Test
